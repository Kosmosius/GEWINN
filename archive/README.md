# Historical archive

This directory is immutable evidence: preserved source manuscripts, prior
reports, historical images, and research material. It is not a canonical rules
source and must never be edited in place. Preserve original filenames and
bytes; create new working copies elsewhere when inspection or migration needs
one.

`v2.5-baseline/` contains the designer-authored baseline and its preserved art
assets. `v5-scaffold/` is reserved for a future v5 procedural scaffold import;
no v5 manuscript was present in this bootstrap. `superseded-drafts/` is
reserved for non-authoritative experimental versions. `imported-reports/`
contains prior analyses and consolidated outputs. `source-research/` is for
historical/research source material that is not a rules manuscript.
