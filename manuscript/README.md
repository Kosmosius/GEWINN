# Manuscript workspace

| Location | Purpose |
| --- | --- |
| `source/` | Editable canonical manuscript source. |
| `generated/` | Generated exports and tables; never edit these files directly. |
| `reference/` | Non-authoritative working references and non-source extraction material. |

The working source remains a byte-preserved v2.5 DOCX baseline until a future,
subsystem-by-subsystem migration applies the material in `migration/v2.5-current/`.
The archival original remains immutable in `archive/v2.5-baseline/`.
