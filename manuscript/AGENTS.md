# Manuscript rules

Edit only `manuscript/source/`; `manuscript/generated/` is regenerated output
and `manuscript/reference/` is non-authoritative. Read the root `AGENTS.md`,
`docs/CANONICAL_SOURCES.md`, and the migration bundle before changing rules.

Preserve v2.5 voice, attribution, front matter, and discrete-wound material.
Do not apply a migration decision, resolve a TODO, or copy values from a table
by hand unless the work is explicitly scoped and backed by `data/`.
